Feed-like App

Feed-Like App is a dynamic, single-page React application that mimics a social media feed. Users can create, edit, and delete posts and comments in real time without page refreshes. The app leverages React Router for seamless navigation and Tailwind CSS for a responsive, modern UI. While data is currently managed via local state (simulating API behavior), the design is built to scale with future backend integration. .
